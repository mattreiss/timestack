## Commands

**Install Dependencies**

```console
yarn
```

**Start Preview**

```console
yarn start
```

**Render video**

```console
./timelapse.sh -D <directory> -FPS <fps> -S <stack_length>
```

**Upgrade Remotion**

```console
yarn run upgrade
```
